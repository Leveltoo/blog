# 你好，世界

欢迎来到我的博客！这是我使用 Next.js 搭建的个人博客网站。

## 为什么选择 Next.js？

Next.js 是一个功能强大的 React 框架，它提供了：

- **服务端渲染 (SSR)** - 更好的 SEO 和首屏加载速度
- **静态站点生成 (SSG)** - 构建时生成页面，部署简单
- **文件系统路由** - 直观的路由配置
- **TypeScript 支持** - 类型安全

## 博客功能

这个博客包含以下功能：

1. 📝 Markdown 文章支持
2. 🏷️ 标签分类系统
3. 🔍 文章搜索
4. 🌙 暗黑模式
5. 🎨 作品集展示
